# LivanCode E-Commerce Platform

A full-stack e-commerce application built with Vue.js, Java Spring Boot, and PostgreSQL.

## Project Structure

```
LivanCode/
├── frontend/          # Vue.js application
├── backend/           # Java Spring Boot API
└── docs/              # Documentation
```

## Tech Stack

- **Frontend**: Vue.js 3, Vue Router, Pinia (state management)
- **Backend**: Java, Spring Boot, Spring Data JPA
- **Database**: PostgreSQL
- **Tools**: Maven, npm, Docker

## Getting Started

### Prerequisites
- Node.js (v16+)
- Java 11+
- PostgreSQL 12+
- Maven 3.6+

### Installation

#### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

#### Backend Setup
```bash
cd backend
mvn clean install
mvn spring-boot:run
```
